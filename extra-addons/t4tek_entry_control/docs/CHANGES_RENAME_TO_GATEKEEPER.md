# Rename UI module to Gatekeeper

- Renamed the Odoo app display name from `T4TEK Entry Control` to `Gatekeeper`.
- Renamed the root menu from `Entry Control` to `Gatekeeper`.
- Renamed user-facing labels such as Settings, Administrator group, Scheduled Action, and model descriptions to Gatekeeper.
- Kept the technical addon name, model names, XML IDs, API routes, config keys, and database tables unchanged to avoid migration risk and keep the Controller integration compatible.
