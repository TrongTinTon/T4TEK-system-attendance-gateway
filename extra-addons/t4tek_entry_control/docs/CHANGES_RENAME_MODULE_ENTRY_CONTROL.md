# Rename module display name

Renamed the Odoo module display name from `T4TEK Attendance Gateway` to `T4TEK Entry Control`.

Reason: the current module is not only an attendance gateway. It manages controllers, ZKTeco devices, employee sync status, attendance logs, and scheduled/manual creation of Odoo Attendances.

Technical module name remains `t4tek_entry_control` to avoid migration or dependency issues.
